Sound Examples

This webpage provides supplementary materials for the article titled "An Adaptive Wavelet-Based Algorithm for Assessing the Quality of the Attack Transients in Non-Percussive Instruments Sounds", submitted to the International Conference on Digital Audio Effects (DAFx) 2025. These audio samples are provided to enhance understanding of the methods and results discussed in the paper.

The full sound dataset can be downloaded here. The dataset includes a zip file with two folders: one containing good attacks and another containing bad attacks. Additionally, a CSV file is provided with annotations on the different attack errors associated with the file names of each bad attack.
